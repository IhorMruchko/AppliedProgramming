=========================ENG=========================
To test this practice edit testInput.txt file.
File can be split by groups of 3 lines.
Each group is strictly ordered:
    first line - is the name of the test function. These names you may find in the Test class of the main.py file.
    second line - is the input data to test.
    third line - is the expected data.
WARNING: All values must be written as python code style.

To add test you have to add test method to the Test class of the main.py file.
Name this method like 'test_some_another_words_to_the_method(self)' and copy name to the testInput.txt file.
You may insert this line wherever you want, but next lines must be strictly after it.
Also, you may use 'target:' keyword to define this line, but still order must be followed.

Next line can be started with "data:" keyword. After that write in python code style value to test.
In the end write expected result with optional keyword 'expected:'.

=========================УКР=========================
Для того, щоб тестувати цю практичну, використайте файл testInput.txt.
Файл може бути поділений на групи по 3 рядочки.
Кожна група є впорядкована:
    Перший рядок - вказується ім'я функції тестування. Ці назви відповідають назвам методів класу Test із файлу main.py.
    Другий рядок - вказується тестову вибірку.
    Третій рядок - вказується очікуваний результат.
ПОПЕРЕДЖЕННЯ: Всі значення повинні бути написані у стилі мови програмування python.

Для того, щоб додати тест, потрібно у класі Test файлу the main.py додати метод.
Назва цього методу повинна починатись із слова 'test' та бути використаною першим рядком групи файлу testInput.txt.
Вставити назву можна у будь-яке місце у файлі, проте після цього рядка повинні йти рядки тестової вибірки та
очікуваного результату саме цього тесту.
Також можна використати ключове слово 'target:' для визначення рядка, проте послідовність все одно повинна бути
дотримана.

Наступний рядок може починатись із ключового слова "data:".
Групу закриває рядок із ключовим словом 'expected:'.
